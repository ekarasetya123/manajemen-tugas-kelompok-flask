Tentang Aplikasi

Nama Aplikasi: Manajemen Tugas Kelompok

Tujuan: Aplikasi ini dibuat untuk memenuhi tugas mata kuliah Pengantar Pemrograman. Bertujuan membantu ketua kelompok mengelola pembagian tugas per mata kuliah, melacak deadline, dan memantau progres pengerjaan oleh anggota kelompok.

Dibuat oleh: Eka Prasetya Wardana, NIM 301250023

Teknologi yang digunakan:

  • Backend: Python Flask, Flask-Login, Flask-SQLAlchemy, Flask-Migrate
  • Database: SQLite (default), dapat diganti ke MySQL/PostgreSQL melalui variabel lingkungan DATABASE_URL
  • Frontend: HTML5, CSS3 (custom), JavaScript minima (Chart.js untuk diagram)

Aplikasi ini dilisensikan di bawah Lisensi MIT.
